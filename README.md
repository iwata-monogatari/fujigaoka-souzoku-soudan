# fujigaoka-souzoku-soudan
fujigaoka-souzoku-soudanは、磐田市を中心に、相続不動産・空き家・実家じまいに悩む方へ向けた相談サイトです。介護事業と不動産事業を行う富士ヶ丘サービス株式会社が、相続発生前後の不安、名義変更後の活用、売却、片付け、管理、住み替えまでを整理し、地域の事情に寄り添って案内します。親の住まいをどうするか、兄弟間で何を確認すべきか、売る前に調べることは何かを、専門用語をできるだけ避けて解説。家族で話し合うきっかけとなる情報と、初回相談・訪問査定・公式LINEへの導線を分かりやすく設計し、放置されがちな不動産の問題を早めに相談へつなげることを目的としています。地域密着の視点から、無理な売却ではなく、選択肢を一緒に考える入口として機能させ、安心して相談できるよう丁寧に支援します。
